CHAPTER 00

マインドセット

このマニュアルを読む前に、まずこの話を聞いてほしい

人間の脳は、30秒以上同じ動画を見続けることができない。

それ以上は「飽きる」——もっと正確に言うと、「拒絶」する。

なぜか。脳のつくりを考えれば、すぐ分かる。

脳は体重の約 2% しかない。 なのに消費エネルギーは全体の 20% を占める。

つまり脳は「小さいのに、燃費が異常に悪い臓器」だ。

だから脳は常に、無意識でこう考えている。

「できるなら、余計な負荷はかけたくない」

考えること、判断すること、感情を動かすこと——これは全部エネルギーを使う。
これは怠けではなく、生き残るための設計だ。
同じ負荷が続くと脳は省エネモードに入る。この省エネモードに入った瞬間、人は「つまらない」と感じる前に、静かに離脱する。

「じゃあ、30秒以上の動画は誰も見れないじゃないか」

そう思うよね。私もそう思った。

でも——30秒以上見せる方法が、ある。

THE ANSWER

「脳の各部位に、正しい順番で、30秒ずつ見せること」

脳は1つの塊ではない。その中に、それぞれ異なる役割を持った部位が集まって「脳」を形成している。
同じ部位を30秒以上使い続けると疲れる。でも、30秒ごとに違う部位を刺激していけば——脳にとって、常に新鮮な30秒が続いていることになる。

これを考えたとき、私は「新しい発明だ」と思った。

でもよく考えたら、世の中にすでに当たり前のように存在していた。

それが「構成」だ

起承転結 / 三幕構成 / シンデレラ構造 / キャンベルの法則

構成とは「脳のどの部位に、どんな順番で、どんな情報を与えるか」を誰でも使えるようにテンプレート化したものだ。
だから起承転結をただ表面的に真似しても、うまくいかない。脳への刺激の順番を理解していなければ、形だけのコピーになってしまう。

このマニュアルでは

構成・エンタメ・面白さの本質を、脳科学の観点から説明する。

表面的な「型」ではなく、「なぜその型が機能するのか」を理解することで、
どんな動画でも面白さを設計できるようになることが目的だ。

MINDSET ADD-ON

外部流入時代に必要なのは、
圧倒的なコンテンツクオリティだ

これまで日本のYouTubeでは「外部流入はNG」とされてきた。
理由は単純で、外部から来た視聴者は維持率が下がり、エンゲージメントが悪化するからだ。

しかし韓国・アメリカでは、外部流入を積極的に活用しているチャンネルが多い。
BreakingDown・ラストコールなどを手がける溝口氏のチームも、ショート動画をX・TikTok・Instagramに積極展開している。
Netflixも同様の戦略を取っている。

なぜ彼らは外部流入を取り込めるのか

「誰が見ても面白い」レベルまでコンテンツを仕上げているから。

外部から来た視聴者がそのまま引き込まれる。だから維持率が落ちない。
コンテンツのクオリティがそのまま「外部流入への耐性」になる。

今後、各チャンネルがさらにメディア化されていくとき——テレビ出演・クライアント案件・他SNSからの流入——外部からの視聴者は必ず発生する。
そのときファンも外部視聴者も、両方が面白いと思える動画でなければ意味がない。

このマニュアルを作っている理由

外部流入時代で戦えるのは、圧倒的なコンテンツクオリティを持つチャンネルだけだ。

溝口チームのような戦略を将来とるためには、まず「誰が見ても面白い」動画を作れる状態が前提条件になる。
そのための動画クオリティを上げることが、このマニュアルの目的でもある。

CHAPTER 01

脳の構造と30秒理論

「なぜこう編集するのか」はすべて脳の仕組みから来ている

人間は、動画の「面白さ」を自分では判断できない。

人体を支配しているのは脳と脊髄だ。そこから張り巡らされた神経が、全身の動き・感情・判断のすべてをコントロールしている。
つまり「面白い」「もっと見たい」という感覚も、脳の各部位が反応した結果として生まれている。本人が意識して選んでいるわけではない。

⚠ 抽象思考で作ると、こうなる

「面白いとは何か」という感覚ベースで動画を作ると、個人の感性・流行・大衆の変数に依存してしまう。
これは毎回ゼロから「面白さ」を発明しようとするのと同じで、再現性がない。

✓ 正しい思考の順序

STEP 1
脳の部位への
刺激を設計する
数百万年変わらない論理
+
STEP 2
あなたの感性を
最後に加える
個性・スタイル・視点
=
OUTPUT
再現性のある
面白い動画

起承転結・三幕構成・シンデレラ構造・キャンベルの法則——これらはすべて「脳の部位に刺激を与える順番」を体系化したものだ。
だから次のPART 1は、このマニュアルで最も重要な項目になる。

PART 1

脳の5段階処理

このマニュアルで最も重要な概念。すべての編集技術はこの5段階に基づいている。

脳には、はっきりしたルールがあります。

軽い脳
→
重い脳

の順番でしか動きたくない…

この順番を無視すると、どんなに良い話でも脳は途中で離脱します。

STIMULATION ORDER — 刺激する順番

①
扁桃体
感情を動かす / 自分ごと化 / フック
▼
②
大脳辺縁系
ドーパミン / 続きを見たくなる / 期待感
▼
③
ミラーニューロン
当事者になる / 共感 / 没入
▼
④
前頭前野
論理・理解 / なるほど / 情報処理
▼
⑤
海馬
長期記憶 / 人生を変える / 行動変容
GOAL

この順番で刺激すること＝「面白い動画を作ること」。
5段階すべてに届いた動画だけが、記憶に残り、人を動かす。

※ 脳はエネルギー消費が軽い部位から重い部位の順で処理する。この流れに逆らうと視聴者は離脱する。

① 扁桃体 — 感情と自分ごと化

役割：「この動画は自分に関係ある」と判断させる。怖い・悲しい・驚きなどの感情を処理する最初のフィルター。

ここを通過しないと、脳は「自分に関係ない」と判断して情報を遮断する。動画の冒頭で必ずここを刺激する。

刺激する素材：強い感情の表情 / 葛藤 / 弱さ / 共感できる状況

動画を再生した瞬間、脳は無意識にこう見積もります：

軽く見れる？
考えさせられる？
感情まで動かされる？

この見積もりより重い負荷がいきなり来た瞬間、人は疲れて離脱します。
（サムネ→冒頭での離脱が起きる理由は、ここで説明できます）

👉 視聴維持率が落ちる最大の理由は、内容の善し悪しというより、脳の"想定負荷"と"実際の負荷"がズレたことです。

② 大脳辺縁系 — 続きを見たくなる

役割：ドーパミン（期待・報酬）を生み出す。「どうなるんだろう」「この先が気になる」という感覚を作る。

⚠ 最初に刺激してはいけない。先に刺激すると「刺激だけを求める状態」になり、長尺動画を見なくなる。

刺激する素材：予測できる流れ / 問いの設置 / 結果の引き延ばし

③ ミラーニューロン — 当事者になる

役割：他者の行動・感情を「自分のこと」として体験させる。「この人の気持ちが分かる」「自分と似ている」と感じた瞬間に刺激される。

ここが刺激されると視聴者は「見ている人 → 物語の当事者」に変わる。

刺激する素材：失敗・挫折を語る言葉 / 苦労している映像 / 過去の写真・昔の映像

④ 前頭前野 — 論理・理解

役割：「なるほど」という論理的な理解を担当する。感情が動いた後にやっとここが働く。

最初に「説明」から始めると、ここしか刺激できず感情が動かないまま終わる。必ず感情の後に届ける。

刺激する素材：整理された情報 / 構造的な説明 / データ / まとめ

⑤ 海馬 — 記憶への定着

役割：情報を長期記憶として保存する。ここまで刺激が届いた動画だけが「記憶に残る動画」になる。

ショート動画が記憶に残らないのは、①②だけを刺激してここに届かないからだ。長尺ドキュメンタリーが「人生を変える」と言われるのは、5段階すべてを刺激するからだ。

刺激する素材：感情を伴った体験の積み重ね / 物語の結末 / 本質的なメッセージ

PART 2 — 各部位の作り方

① 扁桃体の刺激 — 冒頭フックの設計

動画の最初に必ずやること

⚠ 最も多い誤解

「喧嘩・炎上・衝撃映像」を冒頭に入れても、長期的なファンにはならない。

確かにバズる。再生数も取れる。でもそれはドーパミン刺激であって、扁桃体刺激ではない。
ドーパミン目的で集まった視聴者は次の刺激を求め続けるだけで、チャンネルのファンにはならない。
むしろブランディングを傷つけ、マイナスに働く可能性の方が高い。

扁桃体刺激の本質は「この動画は自分ごとだ」と思わせることだ。
視聴者の脳に「これを見ないと、自分は損をする」「この人の話は自分に関係ある」と判断させること——これがフックの唯一の目的だ。

✅ 扁桃体刺激（ファンになる）

「10年間ずっと後悔してきた」「あのとき諦めなければよかった」

感情が直接動く。見た人は「なぜ？」「この人に何があったの？」と感じ、物語の中に入っていく。長尺への移行にも有効。ショートで扁桃体を刺激した視聴者は、続きを知りたくて長尺を見る。

❌ ドーパミン刺激（消費されて終わる）

「衝撃映像！」「○秒後に驚くことが…」「信じられない結末！」

好奇心は煽れるが感情は動いていない。再生数は取れても、チャンネル登録・マネタイズ・ファン化には繋がらない。むしろディスブランディングのリスクがある。

MEMO — ショートから長尺へ移行させる場合も同じ考え方

ショート動画で扁桃体を刺激できていれば、視聴者は「この人のことをもっと知りたい」という動機で長尺に流入してくる。
ドーパミン刺激のショートで集めた視聴者は、長尺を見ない。「次の短い刺激」を求めるだけだからだ。

② 大脳辺縁系の刺激 — 「続きを見たい」状態を作る

ドーパミンを正しく使う

大脳辺縁系はドーパミン（期待・報酬）を生み出す部位だ。ドーパミンは「予測と期待」によって発生する——「この先どうなるか」を予測した瞬間に興味が生まれ、その結果が確認できるとき人は動画を見続ける。

ドーパミンの発生構造
予測させる
「次どうなる？」
→
引き延ばす
答えを見せない
→
結果を見せる
ドーパミン放出

この構造を維持するために必要なのがブリッジ（文脈のつなぎ）だ。シーンが変わるとき、視聴者が「なぜ場面が変わったのか」を考えなければいけない状態になると、前頭前野を使わせてしまう。前頭前野を使わせると動画は疲れる。

ブリッジありの例

手帳の話 →「実際に見せてもらえますか？」→ オフィスへ移動 → 実物の手帳を見せる

「じゃあ実際に見せてもらえますか？」の一言があるだけで、視聴者はオフィスに移動することを自然に理解できる。考えなくていい。
ブリッジなしの例

手帳の話 → 突然オフィスの映像 →「？？？」

視聴者は「なぜ？」と考える。前頭前野を使う。テンポが壊れ、離脱が起きる。

ブリッジになるもの：インタビューの質問 / ナレーション / テロップ / 移動カット / Bロール。これらを使って「次のシーンが予測できる状態」を常に維持する。

③ ミラーニューロンの刺激 — 共感を設計する

視聴者を「物語の当事者」にする

扁桃体で「自分に関係ある」、大脳辺縁系で「続きが気になる」状態になった視聴者に、次に届けるのは共感だ。共感が生まれた後にはじめて前頭前野が働き、「この人の話をもっと聞きたい」という状態になる。共感 → 思考の順番が重要。

① 言語による共感（聴覚）

言葉で共感を生む

失敗・苦労・挫折・弱さを語る言葉が、視聴者の過去の経験と結びついたとき共感が生まれる。

例：「昔は全然うまくいかなくて…」「正直あの時は本当に辛かった」

重要：抽象的な話ではなく具体的な実体験であること。人は抽象に共感しない。
② 視覚による共感（映像）

映像で共感を生む

人は言葉より映像の方が共感しやすい。「昔は大変だった」と言うより、昔の写真・映像を見せる方が共感は強くなる。

使える素材：苦労・失敗している姿 / 辛そうな表情 / 過去の写真・映像 / リアクション

ミラーニューロンは編集だけでは作れない。素材はインタビュー・過去写真・Bロールの中にすでに存在している。編集者の仕事は ①共感できる素材を見つけ、②正しいタイミングに配置し、③共感の流れを壊さないように編集すること。

PART 3 — 30秒理論

30秒理論は「最初の30秒」の話ではない。動画全体が「30秒の塊」の連続で構成されているという構造の話だ。

脳は体の中で最もエネルギーを消費する器官だ。同じ脳の部位を30秒以上使い続けると省エネモードに入る——つまり集中力が切れて離脱が起きる。だから編集は「30秒ごとに新しい刺激（＝新しい脳の部位への切り替え）を届ける」意識で作る。

30秒の塊の構造（動画全体で繰り返す）
扁桃体を刺激
感情・自分ごと化
→
大脳辺縁系へ
続きを見たくなる
→
問いを残す
次の塊へ
→
繰り返す
最後まで

この構造が途切れた瞬間に離脱が起きる。編集とはこの連鎖を動画全体に設計することだ。

PART 4 — エンタメ化

動画には「話の軸（伝えたい内容）」がある。しかしそのまま説明すると前頭前野ばかりを使うため、人は飽きる。 「エンタメ化」とは、前頭前野の情報を大脳辺縁系で楽しめる形に変換することだ。

エンタメ化の構造（ジェットコースター型）
話の軸
前頭前野
→
エンタメ化
寸劇・比喩・小話
→
話に戻る
前頭前野
→
またエンタメ化
繰り返す

エンタメ化の例：寸劇 / 比喩・たとえ話 / ナレーション / 小さなストーリー / 意外な切り口。これらを「遠回り」として挟み込み、また本題に戻る。この繰り返しが「飽きない動画」を作る。

REFERENCE VIDEO — エンタメ化の実例

泣いている・悔しがっている・葛藤している表情のクローズアップ / 「なぜ○○したのか？」という問いかけテロップ / 物語の中で一番感情が高まる場面の断片（答えは見せない）/ 弱さや失敗をさらけ出した一言
ショート動画は主に「大脳辺縁系」だけを刺激する構造。刺激は強いが海馬まで届かないため記憶に残らない。「去年見たショート動画で一番記憶に残っているものは？」と聞かれてほとんどの人が答えられないのはこのためだ。

長尺動画は前頭前野・海馬まで刺激を届けられる。だから「考えさせて記憶に残す」ことが得意なジャンルだ。ドキュメンタリーが「人生を変える」と言われる理由はここにある。
各30秒の終わりに「問いを未解決のまま次へ渡す」ことが基本。「この人は諦めなかった——でも、なぜ？」「そしてその決断が、後に大きな結果をもたらすことになる」など、答えを引き延ばすことで視聴者を引き続ける。
CHAPTER 05

ワークフロー

Flip The Bird の編集の手順（この順番で必ずやる）

各ステップは独立しているようで連動している。「アバン」「インサート」「BGM」は独立した技術ではなく、このワークフローの中にある特定のステップで使うツールだ。

01
素材整理・書き起こし
撮ってきた映像を全部Premiereに読み込む。インタビューは全部文字に書き起こす。使えそうな発言に色をつけておく。
02
1語ゴールの設定
「この動画を見終わったとき、見ている人に何を感じてほしいか」を1つの言葉で決める。迷ったときはこれに戻る。
03
構成プロット作成 + アバン設計
発言を並べて物語の骨格を作る。「成長の物語（キャンベル型）」か「逆転の物語（シンデレラ型）」かを選ぶ。骨格が決まったら、アバン（冒頭のつかみ）を設計する。

STEP 03 詳細 — アバン設計

動画のタイトルが出る前に流れる「つかみ」の作り方

感情フック
→
問いの設置
→
タイトル
→
本編スタート
アバンの目的
見ている人に「最後まで見ないと答えが分からない」と感じさせること。「気になる！」という気持ちを最初に作る。
答えを見せるな
アバンでは絶対に結末を見せない。感情が一番高まる場面を少し見せながら、「なぜこうなったの？」という疑問を作る。
長さの目安
30秒〜90秒が目安。長すぎると見ている人が疲れてしまう。短すぎると疑問が生まれない。60秒くらいがちょうどいい場合が多い。
  • 強い感情のある素材から始まっているか
  • 「なぜ？」「どうなった？」の問いが立っているか
  • 答え（結末）を見せていないか
  • タイトルに繋がるテンションがあるか
04
BGMセレクト
感情の流れを先に決める。BGMを先に入れてから、それに合わせてインサートを選ぶ。

STEP 04 詳細 — BGM・音の使い方

🎵
音は感情の下地
BGMは映像より先に「次はこんな気持ちになるよ」と伝える。人は映像を見るより0.2秒早く、音で次の感情を準備している。
📉
引き算の音設計
一番盛り上がる場面ほど音楽を減らす。「無音」は最も強い音楽。ずっとBGMを流し続けると感情がフラットになってしまう。
🔊
MAの重要性
MA（音楽・効果音・セリフのバランス調整）をきちんとやること。どんなに映像が良くても、音のバランスが悪いと素人っぽく見えてしまう。
BGMは映像のカットが変わるタイミングで入れない。映像の「雰囲気・感情」が変わるタイミングで入れる。切るときは台詞が終わった直後か、音楽のフレーズの区切りで切る。フェードアウトはどうしようもないときだけ。
環境音（その場所の空気音）は「本当にそこにいる感じ」を作る。インタビュー中も小さく環境音を流すことで、「この場所にいる感」が出る。撮影のとき、必ず環境音だけを録音した素材を用意すること。
編集の順番は「BGMを先に決める → BGMに合わせてインサートを選ぶ → チャプター（場面の区切り）を決める」。音楽のリズムと感情の流れを先に決めると、インサートの選び方が自然と決まる。
05
ラフカット → ファインカット + インサート
まず「言いたいことが全部入っているか」だけを確認するラフカット（大雑把な編集）を作る。次にリズムと感情を整えるファインカット（細かい仕上げ）をする。インサートはファインカット時に映像の「体験」として差し込む。

STEP 05 詳細 — インサートの使い方

映像を「補足」に使うな。「体験」に使え

インサートは「説明を分かりやすくする補足映像」だと思われている。それは最も浅い使い方だ。
本来インサートは、視聴者に体験させるための装置だ。

この三層構造が揃うと、視聴者は「動画を見ている」のではなく「物語を体験している」状態になる。その結果、視聴維持率は自然と高くなる。

PART 1 — インサートで「物語」を作る

インサートの究極形は再現ドラマだ。「世界仰天ニュース」などの番組が使う再現VTRは、説明ではなく追体験を作る。視聴者は「情報を受け取っている」のではなく「その場にいる感覚」で見ている。毎回キャスト・ロケを用意するのは現実的でないが、その原理はカットの組み合わせで再現できる。

カットの数で「補足」が「体験」に変わる
❌ 1カットだけ（補足止まり）

「私はお茶を飲みました」

→ お茶の写真を1カット

視聴者の頭の中：「お茶ね、はい」で終わる。情報として処理されるだけ。
✅ 3〜4カット（体験になる）

「私はお茶を飲みました」

→ 急須を置く → お湯を注ぐ → 湯気が立つ → 口に運ぶ

視聴者の頭の中：その場面を「体験」する。ミラーニューロンが刺激される。

インサートは「情報を見せるもの」ではなく「行動を追体験させる装置」だ。脳は物語の形で情報を受け取ると、疲れず・自然な順序で理解でき・感情と一緒に記憶される。だから複数カットで「行動の流れ」を作ることが基本になる。

PART 2 — インサートで「世界観」を操作する

インサートには、世界観そのものを変える力がある。「お茶を飲む」という同じ行動でも、どのインサートを選ぶかで視聴者が頭の中で描く物語が完全に変わる。

WORLD A
和室 × おばあちゃん

→ 懐かしさ / 家族 / 温もり / ゆったりした時間

WORLD B
大理石テーブル × 一人

→ 孤独 / 都会 / 洗練 / 何か考えている

WORLD C
山の上 × 景色

→ 達成感 / 自由 / 解放 / 人生の転機

インサートは「情報の補足」ではなく「物語の方向性を決める装置」だ。どのインサートを選ぶかは、視聴者に「どんな世界観を体験させたいか」を決めることと同じだ。

インサートの役割は2種類ある
TYPE A — モノ・風景インサート

世界の「場所」を決める

視聴者が「この動画はどんな世界の話か」を理解する。舞台設定・文脈を作る役割。

例：工場の外観 / 街の風景 / 自室のデスク / 夜の交差点
TYPE B — 表情・仕草インサート

世界の「温度・感情」を決める

視聴者が「この人は今どんな感情の世界にいるか」を感じ取る。視線を落とす / 沈黙 / 口元が緩む / 小さくため息をつく——言葉がなくても感情が伝わる。

例：言葉の前の沈黙 / 目線のゆれ / 話しながら少し笑う

PART 3 — 情報インサート vs 感情インサート

プロの編集者が「どこにどのインサートを入れるか」を判断するとき、最初に考えるのはこの分類だ。インサートには目的が2種類あり、場面によって使い分ける。

情報インサート（前頭前野に届ける）

「何の話か」を理解させる

視聴者が状況・場所・物事を把握するために必要なインサート。理解を助けるが、感情は動かしにくい。

使う場面：話題が切り替わるとき / 場所・人物を紹介するとき / 具体的なモノや手順を見せるとき

例：工場の外観 / 商品のアップ / 地図 / 資料画面
感情インサート（扁桃体・ミラーニューロンに届ける）

「どんな気持ちか」を感じさせる

視聴者の感情・共感を引き出すインサート。情報を伝えるより、その場の「温度」を作ることが目的。

使う場面：感情が動く発言の直後 / 物語のターニングポイント / 共感を深めたい場面

例：表情のクローズアップ / 手の動き / 沈黙中の視線 / 懐かしい写真
プロの配置ルール
情報の後
情報インサートを入れて「理解」させてから、感情インサートで「共感」させる。理解が先にないと感情が動かない場面もある。
感情的発言の直後
感情インサートを優先する。発言の直後に表情や沈黙を入れると、その感情が視聴者に乗り移る。情報インサートを挟むと感情が冷める。
テンポが速い場面
情報インサートを短く刻む。1カット1〜2秒で切り替えることで、脳が考える前に次へ進む感覚を作れる。
感情を深める場面
感情インサートを長めに置く。表情・沈黙・仕草に2〜4秒かけると、視聴者がその感情の中に留まる時間ができる。
  • インサートは1カットで終わっていないか（3〜4カットで体験にできるか）
  • そのインサートは「情報」を届けたいのか「感情」を届けたいのか判断できているか
  • 感情が動く発言の直後に、情報インサートを挟んでいないか
  • モノ・風景（場所）と表情・仕草（温度）の両方が使えているか
  • インタビューで質問している人を映していないか
インタビュー映像の手・目・口のアップを感情インサートとして使う / テロップで内容を情報インサート代わりにする / 過去の写真・昔の映像（クレジット必須）/ アニメーションや図で情報を補足する。

素材がなくても「表情インサート」だけは必ず取れる。インタビュー収録時に「カメラを回したまま沈黙の時間を作る」ことを習慣にすると、感情インサートの素材が自然に増える。
情報インサート：1〜2秒（見れば分かるものは短く）/ 感情インサート：2〜4秒（感情が乗り移るまで置く）/ 世界観を作る風景インサート：3〜5秒（場所・雰囲気を感じさせる時間が必要）。カットを切るタイミングは、話している人の息継ぎに合わせると自然に聞こえる。
06
テロップ・MA・カラコレ
テロップを入れる → 音のバランスを整える（MA） → 映像の色を整える（カラーグレーディング）の順で仕上げる。

3秒ルール — 画角変更の鉄則

「3秒」は最小単位。実際には4つの尺に分解して考える。

カットを切るたびに必ず映像の見え方（画角）を変える。同じ距離・同じ角度の映像を連続して使わない。最長でも3秒に1回は必ず「見え方を変える」。

3〜5秒 INSERT

インサート尺

画角を変えるために差し込む映像クリップ1枚の長さ。3秒未満は短すぎて脳が処理できず、5秒を超えると間延びする。インサートはこの範囲に収める。

15秒 VLOG

VLOG尺 — 人間性が見える雑談

本編に直接関係はないが、その人の人間性・キャラクターが見えるシーン。雑談・移動・何気ない一言など。15秒で収める。ゼロだと人物が平面的になる。

60秒 SCENE

本編シーン尺 — 1つの話題は60秒でまとめる

本編の情報・感情・ストーリーの1単位は60秒以内にまとめる。60秒の中に30秒ルールが2回発生する——つまり1つの会話の中に必ず1度場面転換が入る。1回の会話で2つの話が展開されるイメージ。

180秒 CORE

訴求パート — 最も伝えたいことは180秒

動画の中で最も伝えたいコアメッセージを届けるパート。ここまでに扁桃体→大脳辺縁系→ミラーニューロンと刺激が積み上がり、前頭前野が受け取れる準備ができているため少し長くても大丈夫。

1語ゴール設計

この動画が終わったとき、見ている人の心に残る気持ちを1語で言えるか？
「感動」「驚き」「やる気」「怖い」「誇り」——これが決まらないと編集は始められない。

01
ゴール
見ている人に「最終的に何を感じてほしいか」を決める。感情を1つの言葉で表す。
例：「この人のようになりたい」という憧れ
02
勝敗条件
物語において何が「成功」で何が「失敗」なのかをはっきりさせる。見ている人が応援できる分かりやすい条件を決める。
例：独立して10年続けられるか否か
03
エンジン
見ている人を次の場面まで引っ張り続ける「疑問」のこと。「この人どうなるの？」という問いが途切れると、見ている人は動画をやめてしまう。
例：「彼は夢を諦めないのか？」という問い
04
キャラクター機能
登場する人それぞれが物語の中でどんな役割を担っているかを決める。
例：主人公・メンター・対照的存在

宇宙兄弟から学ぶ構造

ゴールがはっきりしているから、すべての出来事が「ゴールに近づいているか、遠ざかっているか」で判断できる。見ている人も同じ目線で物語を追える。ドキュメンタリーも同じで、最初に「この人のゴールは何か」をはっきり示すことが大切。
勝敗の条件が具体的なほど緊張感が生まれる。「試験に受かるか落ちるか」は誰でも分かる条件。ドキュメンタリーでも「この人の挑戦が成功するかどうか」をできるだけ具体的に示す。
エンジンは「答えが出るまで見続けさせる問い」。この問いが毎回のエピソードを跨いで視聴者を引っ張る。ドキュメンタリーでは「この人はどうなるのか」という問いを最初に植え付ける。

構成プロット：2つの型

キャンベル型
英雄の旅
普通の日常 → 試練がやってくる → 変わっていく → 元の世界に戻る

主人公が「変化する物語」。挑戦・成長・発見の話に使う。「〇〇が変わった話」に最適。
シンデレラ型
上昇と転落
辛い状況 → うまくいく → また落ちる → 救われる

感情の「振れ幅」が一番大きくなる型。見ている人の共感が一番強く生まれる。「逆転の物語」に使う。

構成を作る手順：① 1語ゴールを決める → ② 成長の物語（キャンベル型）か逆転の物語（シンデレラ型）を選ぶ → ③ 発言・素材をその型に当てはめて並べる → ④ 「次はどうなるの？」という疑問が途切れていないか確認する

CHAPTER 08

キャラクター設計

見ている人を引き込む「人物の描き方」

弱さの設計

人は「完璧な人」には共感できない。弱さ・欠点・悩みがあるから「分かる」と感じ、応援したくなる。撮影する相手の「弱さ」を引き出すことが、共感を作るいちばん大事な仕事だ。

主人公
物語のゴールを追いかける中心の人物。弱さと強さを両方持ち、見ている人が「自分と似てる」と感じられる「普通の人間らしさ」が絶対に必要。
メンター
主人公に気づきや勇気を与える人。動画では「アドバイスをくれる人」「師匠」「先輩」として登場することが多い。
対照的存在
主人公の選択や考え方を際立たせる存在。「この人みたいにはなりたくない」「この人みたいになれるかも」という比較を作る。
試練の提供者
主人公に試練を与える存在（人・状況・社会の仕組みなど）。「悪役」である必要はない。越えなければいけない「壁」であればいい。

キャラクターの引き出し方（インタビュー設計）

「一番つらかったのはいつですか」「やめようと思ったことはありますか」「失敗したときどんな気持ちでしたか」——弱さを話してもらうには信頼関係が必要。インタビュー前にたくさん雑談して、安心してもらうこと。
「それを知ったとき、最初に何を感じましたか」「その瞬間のこと、今でも覚えていますか」——「その時の気持ち」を今の言葉で話してもらうと、インタビューが「場面」として映像になる。
「なぜこれをやり続けているんですか」「お金にならなくてもやりますか」「誰かに止められたらどうしますか」——「なぜやるのか」の答えが純粋であるほど、見ている人の共感は深くなる。
